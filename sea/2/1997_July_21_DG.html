IEEE Ethics-Guidelines Discussion Group: I suggest we take a look at this Draft. It is structured with both main statements of Code Ethical Behaviour, as well as including supporting Guidelines.

Some Questions for consideration:

1. Is this a possible model for creating a set of Guidelines for the IEEE Code of Ethics?

2. As this is an activity in which an IEEE entity (Computer Society) is participating, should this new Code be coordinated with the already established and possibly overriding IEEE Code of Ethics?

Go to this WEB page address for this Draft Code:

http://www.ccsr.cms.dmu.ac.uk/codes/se?draft.html

Walter Elden

The IEEE Ethics Committee created this discussion list to solicit recommendations for creating a Ethics Guidelines document. The document will supplement the IEEE Code of Ethics and serve the education, application and interpretation of the brief 10 statement Code.
