User Tag Help

 

User tags are key words added to an NEB record by other researchers. Much like the social tagging systems used by online services such as del.icio.us and Flickr, user tags allow you to easily locate records you have found useful in the past, and share these resources with others. Click here for more information on user tags.

 

Adding User Tags

 

To begin adding user tags to the NEB, please contact cseplibrary@iit.edu to receive a login name and password. Enter this information in the NEB login page, and click "submit." After you have logged in, you can add tags to records by clicking on the "edit" button that will appear below each record. Scroll down to the to the "User Tags" field on the full record page, and enter the desired key words.

 

When choosing how to tag a record, you may want to keep in mind that the NEB is a community resource for researchers interested in the implications nanotechnology has for society. Ideally, other researchers will use your tags to help find related materials in the NEB. For example, when looking at an article on poverty and nanotechnology, a graduate student finds that someone has added the user tag "nano divide" to the record. By looking up this user tag in advanced search, she would find a number of related articles discussing the potential for nanotechnology to increase the "divide" between rich and poor nations.

 

Please refrain from using this function to make political statements, share opinions, or add irrelevant/offensive comments. The tags you add to records will only be helpful to other researchers if they reflect the content of the cited material. CSEP reserves the right to remove user tags of this nature at its sole discretion.

 

Searching User Tags

 

To conduct a search using a user tag, go to the advanced search page and enter your key terms in the User Tags field. Connect words in a phrase with an underscore (ex. magic_nano). You can see a list of all the user tags that have been recently added to NEB records here.
